pub fn bus_unwatch_name(watcher_id: WatcherId)
Expand description

Stops watching a name.

Note that there may still be D-Bus traffic to process (relating to watching and unwatching the name) in the current thread-default crate::glib::MainContext after this function has returned. You should continue to iterate the crate::glib::MainContext until the GDestroyNotify function passed to g_bus_watch_name() is called, in order to avoid memory leaks through callbacks queued on the crate::glib::MainContext after it’s stopped being iterated.

watcher_id

An identifier obtained from g_bus_watch_name()
